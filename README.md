# 🎓 SkillVista — Developer Learning Platform

A colorful, interactive developer learning platform that helps aspiring engineers navigate their learning journey with AI-powered roadmaps, skill assessments, project ideas, and career tools — all in one place.

## 🎯 What Is SkillVista?

SkillVista is designed for developers at every stage — from beginners choosing their first language to mid-level engineers planning their next career move. It combines curated learning roadmaps, an AI tutor, hands-on project ideas, and a personalized dashboard to make learning structured and motivating.

## ✨ Features

- **Learning Roadmaps** — Visual, step-by-step paths for Frontend, Backend, DSA, ML, DevOps, and more
- **AI Tutor** — Ask questions and get personalized explanations on any topic
- **Project Ideas** — Curated project suggestions matched to your current skill level
- **Skill Assessments** — Test your knowledge with topic-specific quizzes
- **Career Tools** — Resume tips, interview prep, and job search strategies
- **Personal Dashboard** — Track progress, streaks, and completed milestones
- **Responsive Design** — Fully mobile-friendly with a vibrant UI

## 🚀 Getting Started

No build tools needed — runs directly in the browser.

```bash
# Clone the repo
git clone https://github.com/astha9900/SkillVista.git
cd SkillVista/project

# Open in browser
open index.html
# Or serve with any static server:
npx serve .
```
